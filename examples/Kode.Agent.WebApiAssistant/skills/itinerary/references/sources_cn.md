# 行程规划：国内常用可核验来源（v0）

这份列表用于“查证外部事实”时快速选取可靠来源。**只在需要查证时加载**，别把它常驻塞进上下文。

## 交通

- 火车/高铁：`12306.cn` / `kyfw.12306.cn`
- 民航监管/公告（用于政策/通知类）：`caac.gov.cn`
- 航司信息（以具体航司官网/官方公告为准）：各航司官网域名（如 `airchina.com.cn` 等）

## 天气

- 中国气象局/气象信息（偏权威口径）：`cma.gov.cn`
- 城市级预报（以页面标注为准）：`weather.com.cn`

## 汇率

- 中国外汇交易中心/中间价等（看具体需求）：`chinamoney.com.cn`
- 如需“市场现价/参考价”，可用主流金融数据提供方并交叉验证（务必标注查询时间与来源）

## 景点/营业时间/门票

- 以“景区/博物馆/场馆官方站点或官方公众号公告”为准（优先）
- 若官方信息缺失：选择覆盖面大的平台作为线索，但关键事实需二次确认（并明确标注来源属性）

## 政策/公告

- 政府站点：`gov.cn` 及地方政府官网域名
- 交通部门/场馆官方公告：以具体机构官网/公告页为准
