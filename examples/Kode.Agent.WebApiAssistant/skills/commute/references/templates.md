# 通勤方案：模板（v0）

## A. 只问 3 句（缺信息时）

1) 你从哪出发、到哪（如果是多点串联，把点位按顺序发我）？  
2) 你希望 **几点出发/几点到**（或大概时间窗口）？  
3) 你更偏向：**省时间 / 省钱 / 少走路更轻松**？

## C. 风险点写法（示例）

- “早高峰耗时波动大，建议预留 15–25 分钟缓冲（我按你给的到达时间反推）”
- “换乘多：如果带大行李，建议走打车方案更稳”
- “夜间公共交通班次少：建议备一个打车兜底”
