# 酒店预订：国内常用可核验来源（v0）

仅在“需要查证外部事实”时加载这份参考，避免占用上下文。

## 官网与官方口径（优先）

- 酒店/品牌官方站点（用于地址、政策、设施等“规则类信息”）
- 酒店官方客服/官方公告（以页面标注为准）

识别“像官方”的启发式（只作辅助，不保证）：
- 域名与品牌强相关，页面有清晰的企业信息/联系方式
- 有明确的条款页面（隐私/预订条款/取消规则）
- 若无法确认，不要把它当权威结论

## OTA（用于可订性与比价）

不在这里硬编码平台名单（可随业务调整）。策略是：
- 选择覆盖面大、信息结构化的平台做“比价与可订性验证”
- 关键条款（取消/税费/早餐/儿童）以**下单页显示**为准，并保留链接

## 政策/公告（必要时）

- 政府/文旅部门官网：`gov.cn` 及地方政府网站（用于临时政策/公告类信息）
